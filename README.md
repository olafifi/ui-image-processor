# FIFI 图片处理

面向个人工作流的网页端图片处理工具，用于快速整理 UI 示意图素材。它运行在浏览器中，支持导入参考图、抠图、裁剪、格式转换、圆角导出、模板复用和批量重命名。

线上地址：[https://olafifi.github.io/ui-image-processor/](https://olafifi.github.io/ui-image-processor/)

## 核心功能

- 图片导入：支持 PNG、JPG/JPEG、WebP，多图导入和拖拽导入。
- 抠图处理：手动触发自动抠图，浏览器本地模型优先，失败时使用启发式兜底。
- 手动修补：支持点选、反点选、擦除透明、恢复前景、撤销和重做。
- 裁剪与尺寸：支持 1:1、4:3、3:4、16:9、9:16、自由比例；导出尺寸默认跟随裁剪尺寸，也可手动调整。
- 圆角与背景：支持圆角裁图，导出时可选择透明背景或纯色背景。
- 格式转换：导出 PNG 或 JPG；透明背景会锁定 PNG。
- 裁剪模板：可保存当前裁剪和导出设置，并应用到其他图片。
- 批量重命名：可下载命名 CSV，在表格里填写新文件名后再上传 CSV/XLSX，ZIP 导出时应用新文件名。
- 批量导出：支持导出当前图片，或打包下载 ZIP。
- 主题切换：右上角提供 6 套低饱和配色主题，并记住上次选择。

## 使用流程

1. 打开网页，导入或拖入图片。
2. 在顶部切换工作区：
   - 抠图/裁剪：处理透明背景、裁剪比例、圆角和导出设置。
   - 批量重命名：批量编辑导出文件名。
3. 需要抠图时，点击“自动抠图全部”。
4. 需要复用裁剪设置时，在右侧面板保存或套用裁剪模板。
5. 处理完成后导出当前图片，或下载 ZIP。

## 隐私与运行方式

这是一个静态前端应用。图片处理在浏览器本地完成，不需要上传图片到服务器。首次使用自动抠图时，浏览器可能需要下载模型和 WASM 资源，因此第一次运行会更慢。

## 本地开发

```bash
npm install
npm run dev
```

常用验证命令：

```bash
npm test
npm run build
npm run e2e
```

## 部署

项目使用 GitHub Pages 部署。推送到 `main` 后，GitHub Actions 会执行：

```bash
npm ci
npm run build
```

构建时会设置 `GITHUB_PAGES=true`，使 Vite 使用 `/ui-image-processor/` 作为资源 base path。
